# gem_polisher

check repo's Gem version

```
$ cd your-work-dir
$ git clone git@github.com:ryudo/gem_polisher.git
$ cd gem_polisher
$ bundle install # sorry install is still global
$ ./gem_polisher rails
```

## Output

```
+-------+--------+
|       |rails   |
+-------+--------+
|repo-A |5.1.6.2 |
|repo-B |5.2.1.1 |
+-------+--------+
```
